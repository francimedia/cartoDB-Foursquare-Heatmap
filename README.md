#CartoDB Foursquare Heatmap

Create your personal checkin heatmap using CartoDB & Foursquare.

##Workflow
* Login with Foursquare
* Download GeoJSON file
* [Sign up at CartoDB](http://www.cartodb.com)
* Skin your map
* Don't share your map! :-)
